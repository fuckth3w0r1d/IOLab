#### 接口设计实验 - Proteus 设计并仿真

`Project/` 中是 proteus 工程文件，包括各个实验的完整原理设计图，可用于仿真

`src/` 中是各个实验需要的 8086 运行程序，自行编译，例如

```
nasm -f bin lab.asm -o lab1.bin
nasm.exe -f bin lab1.asm -o lab1.bin
```

然后在对应实验的工程中，将 .bin 载入 8086 即可，设置好 8086 的合适内存大小即可仿真

> 实验内容
>
> - 1 利用 8086 直接读取 8 个开关状态，当开关闭合时，对应 LED 灯亮，反之则熄灭。
> - 2 利用 8255A 读取 8 个开关状态，当开关闭合时，对应 LED 灯亮，反之则熄灭。
> - 3 假定输入时钟信号为 1MHz，利用 8253 生成 1KHz 的连续方波和负脉冲信号，分别从计数通道 1 和通道 2 输出到虚拟示波器。
> - 4 假定输入时钟信号为 1MHz，利用 8253、8259A、8 个 LED 灯、1 个数码管等元件，每 1 秒钟控制 1 个 LED 灯亮，并且显示该灯的编号。即：初始所有灯和数字不亮，运行程序后，第 1 秒钟后第一个 LED 灯亮，数码管显示“1”，第 2 秒钟后第一个 LED 灯熄灭，第二个 LED 灯亮，数码管显示“2”，……，直到 8 个灯依次熄灭后就结束。
> - 5 按键与数码管显示：4×4 矩阵键盘通过 8255A 接口，按键值在 7 段数码管上显示；
>
> 周荷琴, 冯焕清. 微型计算机原理与接口技术[M]. 第6版. 合肥: 中国科学技术大学出版社, 2019.
